# unknot - 絡まりをほぐす対話ツール

**unknot（アンノット）** は、ユーザーの漠然とした悩みや「困りごと」をAIとの対話を通じて紐解き、構造化された「問題解決ツリー」へと自動で整理する対話型Webアプリケーションです。

## 🌟 コンセプト
「何に困っているかはわかるけれど、どう整理すればいいか分からない」
そんな心の絡まりを、AIがベテランの相談員のように寄り添い、具体的な原因と解決のゴールへと導きます。

## ✨ 主な機能
- **ダブルAIエージェント・システム**:
  - **対話エージェント (Agent A)**: ユーザーの話を親身に聞き出し、深掘りするためのヒアリングを行います。
  - **構造化エージェント (Agent B)**: 会話の内容をリアルタイムに解析し、問題解決ツリー（なぜなぜ分析形式）を自動生成・更新します。
- **問題解決ツリーの視覚化**:
  - レベル1（困りごとの概要）からレベル5（具体的な案内先・解決策）まで、5段階で問題を分解。
  - AIが提案した「仮説」をユーザーが確認・修正・確定させるインタラクティブなUI。
- **柔軟なカスタマイズ**:
  - AIの役割（相談員、コンサルタント、技術サポーターなど）や、最終的なゴール設定を自由に変更可能。
- **マルチモデル対応**:
  - Google Gemini API、およびローカルLLM（Ollama）に対応。
- **エクスポート機能**:
  - 生成されたツリーをテキスト（樹形図形式）またはJSONとして保存・クリップボードコピーが可能。

## 🛠 技術スタック
- **Frontend**: React (Hooks / Functional Components)
- **Styling**: Tailwind CSS, Font Awesome 6
- **Markdown**: Marked.js
- **API**: Fetch API (Gemini / Ollama)
- **Deployment**: シングルHTMLファイル構成（SPA）

## 🚀 使い方
1. [index.html](cci:7://file:///Users/kazu/unknot.local/index.html:0:0-0:0) をブラウザで開きます。
2. 設定（歯車アイコン）から、AIモデル（Gemini等）とAPIキーを設定します。
3. チャット欄に今の「困りごと」を入力して、対話をスタートしてください。

## 📝 開発の背景
複雑に絡まり合った問題を解決するためには、「言語化」と「構造化」が不可欠です。このツールは、AIが聞き手として言語化を助け、同時にシステムとして構造化を担うことで、ユーザーが次のアクションを明確にできるように設計されました。
